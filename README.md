# FlexCPP™ — Metallized & Specialty CPP Solutions

Product-centric, bilingual (RU / EN) interactive HTML presentation.
No brand-owner names — the focus is entirely on the FlexCPP™ product portfolio.

## Open it
- Double-click **`app/index.html`** (or the root **`index.html`**, which redirects to it).
- Direct language links: **`app/russian.html`** · **`app/english.html`**

## ⭐ Interactive flip cards (slide 5)
Each of the 13 CPP grades is a 3D flip card. Click / tap (or focus + Enter) to flip it
and reveal **real Technical Data Sheet** values transcribed from flexfilm.com:
thickness, SIT, seal strength, density, film structure, key features and applications.

Grades: C-CPM · C-CLL · C-CHL · C-CWL · C-CWU · C-MNC · C-CDF · C-CPR · C-CBP · C-CPT · C-CKP · C-CPA · C-CPL

## Keyboard
| Key | Action |
|---|---|
| → / Space / PageDown | Next slide |
| ← / PageUp | Previous slide |
| Home / End | First / Last |
| L | Toggle RU ⇄ EN |
| F | Fullscreen |
| P | Print / Save PDF |
| Enter / Space (on a card) | Flip the grade card |
